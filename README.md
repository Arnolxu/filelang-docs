# filelang-docs
Documentation for [FileLang](https://github.com/Camroku/filelang-docs). Uses [Retype](https://retype.com) to generate static pages.

## Contibuting
Template for a function:
````md
---
label: func Function
---
# `func` Function

Does something.
+++ Usage
```
func args
```
+++ Example
```
func arg1
```
+++
````
